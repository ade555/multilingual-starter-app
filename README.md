# MyApp

A simple React + Vite starter app with three pages.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Pages

- `/` — Home
- `/about` — About
- `/contact` — Contact

## Stack

- [React 19](https://react.dev)
- [Vite](https://vitejs.dev)
- [React Router v7](https://reactrouter.com)
- TypeScript
